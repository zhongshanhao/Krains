96. 不同的二叉搜索树

Krains 2020-07-17 动态规划数学

# 题目链接

# 动态规划

# 状态定义

令dp[i]为i个结点所能构建的二叉搜索树的个数。

# 状态转移

考虑i个结点时，可以选择m作为其头结点，m属于[1,i]，当其左子树结点个数为j时，除去其头结点，右子树结点个数为i-j-1,此时以m为头结点的树的个数为 dp[i]∗dp[i−j−1]dp[i] * dp[i-j-1]dp[i]∗dp[i−j−1] ,也就是其左右子树能构建的二叉搜索树个数的乘积。

我们枚举所有可能的头结点m，将他们求和就能得到dp[i]dp[i]dp[i]，因此得状态转移：

dp[i]=∑j=0j=i−1dp[j]∗dp[i−j−1]dp[i] = \sum_{j=0}^{j=i-1}dp[j]*dp[i-j-1] dp[i]=j=0∑j=i−1​dp[j]∗dp[i−j−1]

# 初始状态

dp[0]=1dp[0] = 1dp[0]=1,即认为树为空时也算一颗树。

代码如下：

class Solution {
    public int numTrees(int n) {
        int[] dp = new int[n+1];
        dp[0] = 1;

        for(int i = 1; i <= n; i++){
            for(int j = 0; j < i; j++){
                dp[i] += dp[j] * dp[i-j-1];
            }
        }
        
        return dp[n];
    }
}

# 数学

上述推导的dp[n]在数学上是卡特兰数CnC_nCn​，即n个节点的二叉搜索树的所有可能形态数为卡特兰数CnC_nCn​为：

C0=1，Cn=2(2n+1)n+2Cn−1C_0=1， C_{n}=\frac{2(2n+1)}{n+2}C_{n-1} C0​=1，Cn​=n+22(2n+1)​Cn−1​

class Solution {
    public int numTrees(int n) {
        long c = 1;

        for(int i = 0; i < n; i++){
            c = 2 * c * (2 * i + 1)/(i + 2);
        }
        
        return (int)c;
    }
}
